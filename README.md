Specifications:
==================

• Name:             Fennec

• Ticker:           FNNC

• PoW Algorithm:    SHA-256

• P2P Port:         8338

• RPC Port:         8339

• Block Size:       4MB

• Block Time:       10 minutes

• Difficulty Time:  1 block

---





• Premine:          2 100 000 FNNC

• Maturity:         100 Confirmations

• Prefix: FNNC adresses start with "F"
